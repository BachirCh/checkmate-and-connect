1. Install: `npm install react-google-recaptcha-v3`

2. Create `lib/spam-protection.ts` with three utilities:

**verifyRecaptcha(token: string, action: string): Promise<{ success: boolean; score: number }>**
- POST to Google reCAPTCHA siteverify endpoint
- Use RECAPTCHA_SECRET_KEY from env
- Return success + score
- Threshold: score >= 0.5

**checkHoneypot(formData: Record<string, any>): boolean**
- Check for a hidden field named `_honey` (or similar)
- If field has value, it's a bot -- return true (is spam)
- Return false for legitimate submissions

**rateLimit(identifier: string, maxAttempts: number, windowMs: number): { allowed: boolean; remaining: number }**
- Simple in-memory rate limiter using Map<string, { count, resetAt }>
- Track by identifier (IP address or similar)
- Return whether request is allowed and remaining attempts
- Note in comment: "In-memory rate limiting resets on server restart. Sufficient for MVP; migrate to Redis (Upstash) for production scale if needed."

3. Create `components/RecaptchaProvider.tsx` -- client component wrapping `GoogleReCaptchaProvider` from `react-google-recaptcha-v3`:
```typescript
'use client';
import { GoogleReCaptchaProvider } from 'react-google-recaptcha-v3';

export function RecaptchaProvider({ children }: { children: React.ReactNode }) {
  return (
    <GoogleReCaptchaProvider reCaptchaKey={process.env.NEXT_PUBLIC_RECAPTCHA_SITE_KEY!}>
      {children}
    </GoogleReCaptchaProvider>
  );
}
```
This provider will wrap form pages (not the entire app) when forms are built in Phase 3.
